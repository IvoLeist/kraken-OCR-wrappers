# Kraken OCR wrappers

Kraken OCR wrappers provides helper scripts, a jupyter notebook, and
a comparison of segmentation outputs for the neural baseline segmenter
vs the legacy box segmenter.

!!! tip "Disclaimer"
    This collection of [kraken](https://github.com/mittagessen/kraken) wrappers is not affiliated with, endorsed by, or maintained by the 
    [kraken project](https://kraken.re).
